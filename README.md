# Giant Project Manager

Módulo de gestão visual de projetos do ecossistema Giant Manager.

## Objetivo

Criar uma plataforma web responsiva, mobile-first e PWA-ready para gestão executiva de projetos, controle de fases, atividades, riscos, pendências, tempo planejado x realizado e geração de status reports automáticos.

## Stack prevista

- Next.js
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma
- PostgreSQL / Supabase
- Vercel
